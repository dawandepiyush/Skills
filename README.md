# Skills

A Claude Code plugin marketplace by [Piyush Dawande](https://github.com/dawandepiyush) based on use cases that I have observed in my work and personal setting.

## Install

```
/plugin marketplace add dawandepiyush/Skills
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [tasklist-checkup](./tasklist-checkup) | Checks up on your task list across 5 dimensions and helps you work through them |
